Предложена имитационная модель движения судна в условиях воздействия внешних факторов, необходимая для 
проверки некоторых положений, связанных с использованием нейронных сетей для модели счисления. Рассмотрена 
возможность применения методов оптимизации для параметрической идентификации модели судна. Приведены 
результаты экспериментальных исследований для проверки адекватности имитационной модели.  
Ключевые слова: счисление пути судна, внешние факторы, имитационная модель, параметрическая идентификация. 
In the article a simulation model of vessel’s motion under external factors influence conditions is considered. This model is 
necessary for the inspection of some matters related with the use of neural networks for vessel’s dead reckoning. The possi-
bility of the optimization methods’ implementation for vessel’s model parametric identification is considered. The experi-
mental results to check imitation model validation are shown. 
Keywords: vessel’s dead reckoning, external factors, imitation model, parametric identification.
