МОДЕЛИРОВАНИЕ НЕОДНОРОДНОГО ПОКРЫТИЯ УПРУГОЙ ПЛАСТИНЫ
С ОПТИМАЛЬНЫМИ ЗВУКООТРАЖАЮЩИМИ СВОЙСТВАМИ

Рассматривается задача об определении законов неоднородности покры
тия плоской упругой пластины, обеспечивающих наименьшее отражение
при заданном угле падения плоской звуковой волны. На основе решения
прямой задачи построен функционал, выражающий интенсивность отраже
ния, и предложен алгоритм его минимизации. Получены аналитические вы
ражения, описывающие механические параметры неоднородного покрытия.

Изменение звукоотражающих характеристик тел можно осуществить с помощью покрытий в
виде непрерывнонеоднородного упругого слоя. Исследовалось рассеяние звуковых волн плос
кими, цилиндрическими и сферическими упругими телами с неоднородными по толщине по
крытиями [1–5]. Непрерывнонеоднородное покрытие можно моделировать системой тонких
однородных упругих слоев, имеющих разные значения механических параметров (плотности и
упругих постоянных) [6].

Настоящая работа посвящена нахождению законов неоднородности покрытия упругой пла
стины, которые обеспечивали бы требуемое звукоотражение.

Отражение и прохождение звука через плоскую однородную изотропную упругую пластину
исследовалось во многих работах (см., например, [7]). Задача об отражении и преломлении
плоской звуковой волны неоднородным упругим плоским слоем решена для изотропного слоя
[8], для трансверсальноизотропного слоя [9], для слоя с анизотропией общего вида [10], для
термоупругого слоя [11].

Рассматривалось восстановление характеристик упругого слоя, произвольно меняющихся по
глубине [12]. Были решены задачи о восстановлении свойств изотропного и ортотропного неод
нородных по толщине слоев по известному полю смещений на границе слоя при анализе уста
новившихся колебаний [13–15]. Определены линейные законы неоднородности плоского упру
гого слоя, имеющего наименьшее отражение при заданном угле падения плоской звуковой вол
ны [16].
