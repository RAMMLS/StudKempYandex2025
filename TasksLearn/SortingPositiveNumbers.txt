Эта программа выполняет роль посредника между клиентским приложением и http сервером, обрабатывая данные, полученные в формате json. Сначала - получаем url адрес сервера,
номер порта и два числовых параметра. 
Затем формирует http get запроск указанному серверу, передавая параметры a && b. Полученный json ответ конвертируется в python список.
Далее программа фильтрует этот список, оставляя только положительные числа и сортируя их в порядке убывания.

Алгоритмическая сложность этой программы определяется в основном операциями с данными, полученными от серера.
Сортировка списка чисел имеет сложность 0(N log n), где N - количество чисел в ответе сервера.
Фильтрация списка положительных чисел выполняется за 0(N). Следовательно общая сложность алгоритма составляет 0(N log N).
